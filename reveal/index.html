何為電腦?

電腦是一種電子設備，在指令的控制下進行操作可以接受數據（輸入）在內存中，根據指定的條件處理數據規則，產生信息（輸出）並存儲信息以備將來使用

Central Processing Unit(CPU)

是電腦的主要裝置之一，功能主要是解釋電腦指令以及處理電腦軟體中的資料。電腦的可程式化性主要是指對中央處理器的編程。

CPU由三個主要部分組成：

Arithmetic logic unit(算術邏輯單元)

Control Unit(控制單元)

Registers(暫存器)

Arithmetic logic unit(算術邏輯單元)

算術邏輯單元（ALU）：執行所有算術和邏輯運算

算術計算:例如加法，減法，乘法和除法

邏輯運算:例如比較數字，字母或特殊字符

Control Unit(控制單元)

控制單元（CU）：控制和協調計算機組件

1.閱讀代碼以執行下一條指令

2.遞增程序計數器，使其指向下一條指令

3.從內存中的單元讀取指令需要的任何數據

4.將必要的數據提供給ALU或寄存器

5.如果說明需要ALU或專用硬件來完成，請說明硬件執行請求的操作

Registers(暫存器)

暫存器:存儲接下來要執行的數據，非常快速的存儲區域

Primary Memory(主記憶)

Random Access Memory(隨機存取存儲器)

Read-Only Memory(只讀存儲器)

隨機存取內存(RAM)

隨機存取存儲器(RAM):是計算機中的一種存儲方案負責臨時存儲數據的系統，以便可以及時進行存儲在需要時由處理器訪問

這代表著一旦關閉存儲設備的電源，數據將被刪除

RAM存儲隨機數據，處理器從RAM隨機訪問這些數據貯存

只讀存儲器(ROM)

ROM(只讀存儲器):ROM是一種永久性的存儲形式

ROM停留不論打開或關閉電源都處於活動狀態

ROM設備不允許修改存儲在其上的數據

Secondary Memory(二級記憶體)

Hard Disk Drive(硬碟)

Solid-state drive(固態硬碟)

硬碟(HDD)

是電腦上使用堅硬的旋轉碟片為基礎

原理是磁頭經過碟片的上方時碟片本身的磁場導致讀取線圈中電氣訊號改變

硬碟的讀寫是採用半隨機存取的方式，可以以任意順序讀取硬碟中的資料，但讀取不同位置的資料速度不相同

硬碟包括一至數片高速轉動的碟片以及放在致動器懸臂上的磁頭

固態硬碟(SSD)

是一種以積體電路製作的電腦儲存裝置

可以用非揮發性記憶體（主要以快閃記憶體中的 NAND Flash））作為永久性儲存裝置，也可以用揮發性記憶體（例如DRAM）作為臨時性儲存裝置

目前由於每單位價格及最大儲存容量與機械硬碟有差距，固態硬碟暫時無法完全取代機械式硬碟

Motherboard(主機板)

是構成複雜電子系統例如電子計算機的中心或者主電路板。

典型的主機板能提供一系列接合點，供處理器、顯示卡、硬碟機、記憶體、對外裝置等裝置接合。它們通常直接插入有關插槽，或用線路連接，主機板上最重要的構成元件是晶片組。

Video card(顯示卡)

是個人電腦最基本組成部分之一，用途是將電腦系統所需要的顯示資訊進行轉換驅動電腦，並提供逐行或隔行掃描訊號，控制電腦的正確顯示，是連接顯示器和個人電腦主機板的重要元件，是「人機對話」的重要裝置之一

Graphics Processing Unit(圖形處理器)

是一種專門在個人電腦、工作站、遊戲機和一些行動裝置（如平板電腦、智慧型手機等）上執行繪圖運算工作的微處理器

圖形處理器

圖形處理器使顯示卡減少對CPU的依賴，並分擔部分原本是由CPU所擔當的工作，尤其是在進行三維繪圖運算時，功效更加明顯。

Internet(網際網路)

是指20世紀末興起電腦網路與電腦網路之間所串成的龐大網路系統

這些網路以一些標準的網路協定相連

它是從地方到全球範圍內幾百萬個私人、學術界、企業和政府的網路所構成，透過電子，無線和光纖網路技術等一系列廣泛技術聯繫在一起

網際網路的起源

1960年代美國聯邦政府委託進行研究，目的是建立容錯與電腦網路的通信

網際網路的前身ARPANET最初在1980年代作為區域學術和軍事網路連接的骨幹

1980年代，NSFNET成為新的骨幹而得到資助，以及其他商業化擴展得到了私人資助，導致了全世界網路技術的發展，以及許多網路的合併

到1990年代初，商業網路和企業之間的連接標誌著向現代網際網路的過渡

網際網路承載範圍廣泛的資訊資源和服務

超文字

全球資訊網

電子郵件

通話

檔案分享

全球資訊網(WWW)

是一個透過網際網路存取的且由許多互相連結的超文字組成的系統

超文字(Hy pertext)

是一種可以顯示在電腦顯示器或電子裝置上的文字

電子郵件(e-mail)

是指一種由一寄件人將數位資訊傳送給一個人或多個人的資訊交換方式，一般會通過網際網路或其他電腦網路進行書寫、傳送和接收信件，目的是達成發信人和收信人之間的資訊互動

基於IP的語音傳輸(VoIP)

是一種語音通話技術，經由網際協定（IP）來達成語音通話與多媒體會議，也就是經由網際網路來進行通訊

檔案分享(File sharing)

是指主動地在網路上(網際網路或小的區域網路)分享自己的電腦檔案

作業系統(OS)

是一組主管並控制電腦操作、運用和執行硬體、軟體資源和提供公共服務來組織使用者互動的相互關聯的系統軟體程式，同時也是電腦系統的核心與基石

作業系統需要處理如管理與組態記憶體、決定系統資源供需的優先次序、控制輸入與輸出裝置、操作網路與管理檔案系統等基本事務

作業系統的型態

不同機器安裝的作業系統可從簡單到複雜，可從行動電話的嵌入式系統到超級電腦的大型作業系統

有些作業系統整合了圖形化使用者介面，而有些僅使用命令行介面，而將圖形化使用者介面視為一種非必要的應用程式

Mechatronics(機電整合學)

是一門利用微電子理論來控制機械裝置的學科，它的技術基礎是來自機械製造和微電子控制，並配合電腦軟體。

機電整合學

機電整合學是用電子硬體、電腦程式，對機械進行控制。當中較為人熟悉的機電一體化成品為機器人，其技術亦會應用到工業生產中所使用的機械臂及生產線自動化上。

不單在工業生產上，在一般我們常接觸的，如小至體內微血管手術機械臂，大至太空梭、太空站都有應該其技術。現時不少固有的產品亦開始加入自動化控制的元件，如汽車及鐵路

傳統機電整合

常見之機電整合多為用電腦輸入數值後交由機器值行，期間尚需要測試、校調，且必需仰賴專業人員與設備，不利於第一線操作人員執行修改或變更數據。

新一代機電整合之概念

新式整合為增加可由機械本身透過第一線人員直接操作機械使其產生可實用、不需多次校調試驗、不需專業人員乃至返回原廠設定等不利於快速作業因素，將數值直接新增保存。

總結

在細查了電腦各個部分的組成及各零件的功能、運作方式後，知道了自己一直以來使用的機器是如何給我們的生活帶來極大的進步，在它給我們生活帶來便利的同時，人們也在試圖使電腦變得更加先進，顯卡，處理器，記憶體等等，個個的變得越來越高階，但跟著進步的速度，汰換速度也逐漸變快。

感想

我們今天使用的網際網路，電腦也佔了不可或缺的一部分，所以這也是為什麼我們打算以電腦當作主題，去了解陪伴在我們生活的高精度計算機，試圖了解它為何如此被重視，以及如何做到那些高精度作業。
