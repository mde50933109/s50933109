Central Processing Unit(CPU)

是電腦的主要裝置之一，功能主要是解釋電腦指令以及處理電腦軟體中的資料。電腦的可程式化性主要是指對中央處理器的編程。

CPU由三個主要部分組成：

Arithmetic logic unit(算術邏輯單元)

Control Unit(控制單元)

Registers(暫存器)

Arithmetic logic unit(算術邏輯單元)

算術邏輯單元（ALU）：執行所有算術和邏輯運算

算術計算:例如加法，減法，乘法和除法

邏輯運算:例如比較數字，字母或特殊字符

Control Unit(控制單元)

控制單元（CU）：控制和協調計算機組件

1.閱讀代碼以執行下一條指令

2.遞增程序計數器，使其指向下一條指令

5.如果說明需要ALU或專用硬件來完成，請說明硬件執行請求的操作

Registers(暫存器)

暫存器:存儲接下來要執行的數據，非常快速的存儲區域

Primary Memory(主記憶)

Random Access Memory(隨機存取存儲器)

Read-Only Memory(只讀存儲器)

隨機存取內存(RAM)

隨機存取存儲器(RAM):是計算機中的一種存儲方案負責臨時存儲數據的系統，以便可以及時進行存儲在需要時由處理器訪問

這代表著一旦關閉存儲設備的電源，數據將被刪除

RAM存儲隨機數據，處理器從RAM隨機訪問這些數據貯存

只讀存儲器(ROM)

ROM(只讀存儲器):ROM是一種永久性的存儲形式

ROM停留不論打開或關閉電源都處於活動狀態

ROM設備不允許修改存儲在其上的數據

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events
